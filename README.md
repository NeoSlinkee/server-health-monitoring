# Server Health Monitoring

This repository contains practical examples and scripts for monitoring Linux servers, with a focus on simple health checks and Zabbix integration.

The goal is to provide lightweight, understandable tools that can be adapted to real environments.

## Contents

- linux-scripts  
  Standalone bash scripts for checking disk usage, load, and critical services.

- zabbix-examples  
  Example Zabbix user parameters and a sample agent configuration snippet.

- docs  
  Short notes on monitoring strategy and basic setup.

## Typical Use Cases

- Quickly checking the health of a server estate.  
- Extending Zabbix with custom checks.  
- Building cron-based or lightweight monitoring where a full stack is not needed.

These are reference examples and should be reviewed before use in production.
